# Python terminal to-do list app 

To run app go to directory and run the .sh or .py file. 


1. Add a new todo

`
$  ./todo.sh add "todo item" 
`

2. Show remaining todos

`
$ ./todo.sh ls                     
`

3. Delete a todo entry

`
$ ./todo.sh del NUMBER             
`

4. Complete a todo entry

`
$ ./todo.sh done NUMBER            
`

5. Show command list

`
$ ./todo.sh help                   
`

6. Show user statistics

`
$ ./todo.sh report                 
`
